Mengapa Bokuchi?

🪶

Ringan

Tauri + Rust membuatnya jauh lebih ringan dari editor berbasis Electron. Penggunaan memori kurang dari sepertiga.

⚡

Langsung menyala

Terbuka dalam waktu kurang dari satu detik. Tulis kapan pun Anda siap.

🌐

Berjalan di mana saja

macOS, Windows, dan Linux. UI dalam 14 bahasa.

🔒

Privat & offline

Tanpa akun, tanpa cloud, tanpa pelacakan. Dokumen Anda tetap di komputer Anda.

Fitur

Pratinjau langsung

Tampilan terbagi: edit di kiri, pratinjau langsung di kanan. Lihat perubahan secara instan saat mengetik.

Matematika & diagram

Render ekspresi matematika KaTeX dan diagram Mermaid langsung di pratinjau. Sempurna untuk penulisan teknis dan akademis.

Toolbar Markdown

Format teks dengan tombol toolbar — tidak perlu menghafal sintaks Markdown. Cocok untuk pemula dan editing cepat.

Pohon folder & outline

Jelajahi file di pohon folder sidebar dan navigasi dokumen dengan panel outline heading.

Editing multi-tab

Buka dan kelola beberapa file sekaligus. Sesi otomatis dipulihkan saat peluncuran ulang.

Sistem variabel

Gunakan placeholder seperti {{author}} yang otomatis diperluas saat ekspor. Mendukung variabel global dan per-file.

Pencarian lintas tab

Cari dan ganti di semua dokumen yang terbuka sekaligus. Temukan apa yang Anda butuhkan secara instan.

Pengalih tema

Pilih dari Default, Dark, Pastel, Vivid, dan Darcula sesuai gaya Anda.
