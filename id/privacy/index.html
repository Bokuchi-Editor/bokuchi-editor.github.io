Kebijakan Privasi

Terakhir diperbarui: 9 April 2026

Kebijakan privasi ini menjelaskan bagaimana situs web Bokuchi (bokuchi.com) menangani informasi saat Anda mengunjungi situs ini.

Analitik

Situs web ini menggunakan Google Analytics 4 (GA4) untuk memahami bagaimana pengunjung menggunakan situs ini. GA4 dapat mengumpulkan informasi berikut:

  • Halaman yang dikunjungi dan waktu yang dihabiskan di setiap halaman
  • Situs web atau sumber rujukan
  • Jenis browser dan sistem operasi
  • Perkiraan lokasi geografis (tingkat negara/wilayah)
  • Jenis perangkat (desktop, seluler, tablet)

Data ini dikumpulkan secara agregat dan tidak digunakan untuk mengidentifikasi pengunjung individu secara pribadi. Google Analytics menggunakan cookie untuk membedakan pengguna unik. Untuk detail lebih lanjut, lihat Kebijakan Privasi Google.

Tentang Aplikasi Bokuchi

Aplikasi desktop Bokuchi tidak mengumpulkan data pribadi apa pun, tidak menggunakan analitik, dan tidak berkomunikasi dengan server eksternal mana pun. Semua dokumen Anda tetap di komputer lokal Anda.

Cookie

Situs web ini menggunakan cookie hanya untuk keperluan Google Analytics. Tidak ada cookie lain yang digunakan.

Layanan Pihak Ketiga

Situs web ini mungkin menautkan ke layanan eksternal seperti GitHub. Layanan-layanan ini memiliki kebijakan privasi masing-masing.

Perubahan Kebijakan Ini

Kebijakan privasi ini dapat diperbarui dari waktu ke waktu. Setiap perubahan akan dipublikasikan di halaman ini.

Kontak

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini, silakan buka issue di repositori GitHub kami.
