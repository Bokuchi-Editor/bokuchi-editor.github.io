隐私政策

最后更新：2026年4月9日

本隐私政策说明 Bokuchi 网站（bokuchi.com）在您访问时如何处理信息。

分析

本网站使用 Google Analytics 4（GA4）来了解访问者如何使用本网站。GA4 可能收集以下信息：

  • 访问的页面及在每个页面上花费的时间
  • 来源网站或渠道
  • 浏览器类型和操作系统
  • 大致的地理位置（国家/地区级别）
  • 设备类型（桌面、移动、平板）

这些数据以汇总形式收集，不会用于识别个人访问者。Google Analytics 使用 Cookie 来区分唯一用户。详情请参阅 Google 隐私政策。

关于 Bokuchi 应用程序

Bokuchi 桌面应用程序不收集任何个人数据，不使用分析工具，也不与任何外部服务器通信。您的所有文档都保存在本地计算机上。

Cookie

本网站仅出于 Google Analytics 的目的使用 Cookie。不使用其他 Cookie。

第三方服务

本网站可能链接到 GitHub 等外部服务。这些服务有各自的隐私政策。

政策变更

本隐私政策可能会不时更新。任何变更都将发布在本页面上。

联系方式

如果您对本隐私政策有任何疑问，请在我们的 GitHub 仓库中提交 Issue。
