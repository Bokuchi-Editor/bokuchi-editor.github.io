なぜ Bokuchi？

🪶

軽量

Tauri + Rust により、Electron ベースのエディタより圧倒的に軽い。メモリ使用量は 1/3 以下。

⚡

即起動

1 秒以内に起動。書きたい時にすぐ書ける。

🌐

どこでも使える

macOS / Windows / Linux 対応。14 言語の UI。

🔒

プライベート＆オフライン

アカウント不要、クラウドなし、トラッキングなし。文書はすべてローカルに保存。

機能

リアルタイムプレビュー

左にエディタ、右にプレビューの分割表示。入力と同時に変更が反映。

数式＆ダイアグラム

KaTeX の数式や Mermaid のダイアグラムをプレビューに直接レンダリング。技術文書や学術論文に最適。

Markdown ツールバー

ツールバーのボタンでテキストを書式設定。Markdown の構文を覚える必要なし。初心者にも最適。

フォルダツリー＆アウトライン

サイドバーでファイルを閲覧し、見出しアウトラインパネルでドキュメントを素早くナビゲート。

マルチタブ編集

複数ファイルを同時に開いて管理。再起動時にセッションが自動復元。

変数システム

{{author}} などのプレースホルダーがエクスポート時に展開。グローバル変数とファイル単位の変数に対応。

クロスタブ検索

開いているすべてのドキュメントを横断して検索・置換。必要な情報を即座に発見。

テーマ切替

Default / Dark / Pastel / Vivid / Darcula から自分好みのテーマを選択。
