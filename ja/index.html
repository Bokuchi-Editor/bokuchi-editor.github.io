なぜ Bokuchi？

🪶

軽量

Tauri + Rust により、Electron ベースのエディタより圧倒的に軽い。メモリ使用量は 1/3 以下。

⚡

即起動

1 秒以内に起動。書きたい時にすぐ書ける。

🌐

どこでも使える

macOS / Windows / Linux 対応。14 言語の UI。

機能

リアルタイムプレビュー

左にエディタ、右にプレビューの分割表示。

マルチタブ

複数ドキュメントを同時に編集。

変数システム

{{author}} などのプレースホルダーがエクスポート時に展開。

検索と置換

ドキュメント内の検索・置換。

テーマ切替

Default / Dark / Pastel / Vivid / Darcula から選択。

設定

タブで分かれた設定画面で、動作や見た目をカスタマイズ。
