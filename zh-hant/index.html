為什麼選擇 Bokuchi？

🪶

輕量

Tauri + Rust 架構，比 Electron 編輯器輕得多。記憶體佔用不到三分之一。

⚡

即時啟動

一秒內啟動。隨時準備好書寫。

🌐

跨平台

支援 macOS、Windows 和 Linux。UI 支援 14 種語言。

🔒

隱私與離線

無需帳號，無雲端，無追蹤。文件始終保存在本機。

功能

即時預覽

分割畫面：左側編輯，右側即時預覽。輸入時立即看到變化。

數學公式與圖表

在預覽中直接渲染 KaTeX 數學公式和 Mermaid 圖表。適用於技術文件和學術寫作。

Markdown 工具列

透過工具列按鈕格式化文字，無需記憶 Markdown 語法。適合初學者和快速編輯。

資料夾樹與大綱

在側邊欄資料夾樹中瀏覽檔案，透過標題大綱面板快速導覽文件。

多分頁編輯

同時開啟和管理多個檔案。重新啟動時自動恢復工作階段。

變數系統

使用 {{author}} 等佔位符，匯出時自動展開。支援全域變數和檔案級變數。

跨分頁搜尋

一次在所有開啟的文件中搜尋和取代。即刻找到所需內容。

主題切換

可選 Default、Dark、Pastel、Vivid 和 Darcula，符合您的風格。
