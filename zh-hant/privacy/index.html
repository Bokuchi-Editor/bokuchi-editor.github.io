隱私政策

最後更新：2026年4月9日

本隱私政策說明 Bokuchi 網站（bokuchi.com）在您造訪時如何處理資訊。

分析

本網站使用 Google Analytics 4（GA4）來了解訪客如何使用本網站。GA4 可能收集以下資訊：

  • 造訪的頁面及在每個頁面上花費的時間
  • 來源網站或管道
  • 瀏覽器類型和作業系統
  • 大致的地理位置（國家/地區層級）
  • 裝置類型（桌面、行動裝置、平板）

這些資料以彙總形式收集，不會用於識別個人訪客。Google Analytics 使用 Cookie 來區分唯一使用者。詳情請參閱 Google 隱私政策。

關於 Bokuchi 應用程式

Bokuchi 桌面應用程式不收集任何個人資料，不使用分析工具，也不與任何外部伺服器通訊。您的所有文件都保存在本機電腦上。

Cookie

本網站僅出於 Google Analytics 的目的使用 Cookie。不使用其他 Cookie。

第三方服務

本網站可能連結到 GitHub 等外部服務。這些服務有各自的隱私政策。

政策變更

本隱私政策可能會不時更新。任何變更都將發布在本頁面上。

聯絡方式

如果您對本隱私政策有任何疑問，請在我們的 GitHub 儲存庫中提交 Issue。
